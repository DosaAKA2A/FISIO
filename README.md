# FISIO — Landing de fisioterapia

Landing page estática para clínica de fisioterapia (Madrid Centro). Un único `index.html` + `/assets`.

## Publicar en GitHub Pages
Settings → Pages → Deploy from branch → `main` / root.

## Estructura
- `index.html` — página completa (HTML/CSS/JS vanilla, sin dependencias)
- `assets/` — vídeo del hero (loop 10 s), póster, fotografías, silueta interactiva, logo e favicon SVG

## Notas
- Reseñas: adaptadas de reseñas reales; sustituir por el texto literal del perfil de Google Business antes de publicar.
- Tarifas orientativas: confirmar con el negocio.
- Vídeo: `assets/hero.mp4` (1440p, CRF 25). Sustituible por metraje propio manteniendo atributos `autoplay muted loop playsinline`.
